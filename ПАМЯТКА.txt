========================================
1. ЧТО ЭТО ЗА ПРОЕКТ
========================================

Веб-приложение на GitHub Pages для прогнозов NBA матчей.
- Ежедневные прогнозы на победителя и тотал
- DeepSeek через OpenRouter API ищет статистику в интернете
- Симуляция 15 матчей мысленно
- Учёт стимула (плей-офф, борьба за место)
- Отображение коэффициентов букмекеров
- Статистика точности прогнозов

========================================
2. КЛЮЧЕВЫЕ ФАЙЛЫ
========================================

ГЛАВНЫЕ СКРИПТЫ:
├── scripts/update_matches.py      ← получает матчи, вызывает DeepSeek, сохраняет прогнозы
├── scripts/check_results.py       ← проверяет результаты прошедших матчей

АВТОМАТИЗАЦИЯ:

├── data/matches.json              ← прогнозы на матчи
├── data/results.json              ← история результатов и статистика

ЗАВИСИМОСТИ:
├── requirements.txt               ← библиотеки: requests, nbainjuries

========================================
3. API КЛЮЧИ (в Secrets GitHub Actions)
========================================

Названия секретов (точь-в-точь):
- ODDS_API_KEY        ← ключ от the-odds-api.com
- OPENROUTER_API_KEY  ← ключ от openrouter.ai (для DeepSeek)

Без них ничего не работает.

========================================
4. ЛОГИКА РАБОТЫ (кратко)
========================================

В 10:00 UTC:
1. Скрипт получает матчи из The Odds API
2. Для каждого матча отправляет запрос DeepSeek
3. DeepSeek ищет в интернете:
   - форму команды (3-2, 4-1 и т.д.)
   - PPG (очки за игру)
   - процент побед
   - H2H (личные встречи)
   - стимул (плей-офф, борьба за место)
4. DeepSeek симулирует матч 15 раз
5. Возвращает прогноз (победитель, вероятность, тотал)
6. Всё сохраняется в matches.json

В 9:00 UTC (или 6:00 UTC для Москвы):
1. Скрипт получает завершённые матчи
2. Сравнивает прогнозы с реальным счётом
3. Обновляет results.json

6. ЧАСТЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
========================================

Проблема: Статистика не отображается (всё "?")
Решение: DeepSeek не вернул данные. Проверить логи Actions.

Проблема: Прогнозы не обновляются
Решение: Проверить, что Actions запускаются по расписанию.

Проблема: Ошибка "ODDS_API_KEY not found"
Решение: Ключ не добавлен в Secrets GitHub.

Проблема: DeepSeek возвращает 50% для всех матчей
Решение: Слишком сложный запрос, упростить промпт.

========================================
7. КАК ЗАПУСТИТЬ ЛОКАЛЬНО (для тестирования)
========================================

1. Установить Python 3.11
2. pip install -r requirements.txt
3. Создать файл .env с ключами:
   ODDS_API_KEY=ключ
   OPENROUTER_API_KEY=ключ
4. python scripts/update_matches.py
5. Запустить локальный сервер для HTML
